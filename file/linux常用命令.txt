touch {filename}  创建文件
mkdir {filename}  创建文件夹
file {filename}  查看文件类型
cat {filename}  查看文件（-n 加行号， -b 非空白行加行号）
less {filename} 查看文件
tail -n 100 {filename} 查看文件后100行（-f 实时刷新）
head -n 100 {filenmae} 查看文件前100行
ps -elf | grep {filename} 查看所有/目标进程
top 查看系统负载/系统情况
df -h 查看内存使用/内存剩余/可用内存
tar -zxvf {filename}  解压
env /pringenv 全局环境变量
echo ${name} 查看某个环境变量
{variable}="mysql" 
export {variable}  设置环境变量
unset {variable} 删除环境变量
chmod 754 {filename}  文件权限/访问/读写
yum list installed  已经安装的包
yum install {filename} 安装包
yum update {filename} 更新包
yum erase/remove {filename} 删除全部/删除软件包保留配置文件和数据文件

vim操作
h：左移一个字符。
j：下移一行（文本中的下一行）。
k：上移一行（文本中的上一行）。
l：右移一个字符。
PageDown（或Ctrl+F）：下翻一屏。
PageUp（或Ctrl+B）：上翻一屏。
G：移到缓冲区的最后一行。
num G：移动到缓冲区中的第num行。
gg：移到缓冲区的第一行。
